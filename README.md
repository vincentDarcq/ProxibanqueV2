Bienvenue sur le site de ProxiBanqueSI ! 
Dans ce readme vous trouverez tout le cheminement des actions réalisables au sein du site ProxiBanqueSI.

     1- Page d'accueil

  Lors de l'ouverture de la page d'accueil, vous pouvez observez une barre de selection en haut de l'écran, 
avec les choix "effectuer une action", "service clients", "team", "contact". Vous pouvez choisir l'action 
voulue en cliquant dessus. Cela vous aménera à l'emplacement concerné sur cette même page.

  "Effectuer une action" vous emmène aux options de gestion client. Il existe 3 options : Mise à jour Client, 
Effectuer un virement, et Consulter mes clients, qui vous affichent toutes 3 votre liste des clients.

  "Service Client" vous emmène également à votre liste de clients présente plus bas sur cette même page.

  "TEAM" vous emmène plus bas où sont listées les personnes ayant travaillé sur ce projet.

  "Contact" est le pied de page où l'on peut trouver les moyens de contacter l'agence. Cette action est momentanément
indisponible, étant donné un serveur mail encore en travaux.

     2- Consulter mes clients

  En cliquant sur 'Consulter mes Clients', 'Effectuer un virement', 'Mise à jour Client', ou 'Service Client', 
vous allez observer votre liste de clients.
  Chaque client possède un encadré où sont affichés son nom, son adresse, son mail. 
Il y a 2 obglets qui s'offrent à vous : +Modifier, et Virement. Ces choix vous emmèront sur une page référencé
du client choisi.

     3- Mise à jour Client

  Lorsque vous visualisez votre liste de clients, vous pouvez cliquer sur l'onglet "Modifier".
Ceci étant fait, vous atterrissez sur un nouvelle page, où les informations relative à votre client sont sélectionnées,
dans des champs de saisie.
  Vous pouvez à présent remplacer n'importe lequel de ces attributs en cliquant sur le champ voulu, en tapant ce
que vous voulez. Toute modification effectuée doit être 'validé', de ce fait cliquer en dessous de vos champs sur 'valider'.
 
  Remarque : si vous êtes sur cette page, mais que vous ne voulez aucune modification, vous pouvez cliquer en haut de
la page à gauche, sur "Retour Accueil". Cela vous raménera à la page d'accueil de ProxiBanque.

     4- Virement

  En cliquant sur l'onglet 'Virement' depuis la page d'accueil, vous atterrissez sur une page contenant des champs
de sélection multiple.



     x- Page d'Erreur

  Si vous êtes amenés à rencontrer une erreur lors de vos actions sur ProxiBanqueSI, ou que le site rencontre un problème, 
vous serez immédiatement redirigé vers une page d'erreur où vous pourrez voir s'afficher "ProxiBanqueSI A RENCONTRE UNE ERREUR".
Vous pouvez revenir à l'accueil en cliquant sur 'Retour Accueil'
